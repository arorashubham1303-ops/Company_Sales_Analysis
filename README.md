# Company_Sales_Analysis
Data analysis project showing case company sale analysis using python, SQL, power Bi
This project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI. The goal is to analyze sales data, generate insights, and present them through dashboards and reports for better decision-making.

📂 Dataset

The dataset contains 20,000 sales transactions with the following fields:

OrderID
Date
Customer
Product
Category
Region
Quantity
UnitPrice
Revenue
PaymentMethod
